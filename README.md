## Online Gift Package 在线礼包
作者：星夜神花
适配优化：羽学
根据config里的【触发间隔】与【触发序列】随机发放【礼包列表】里的物品



## 命令
显示礼包奖励概率表的指令为：/在线礼包

## 版本
v1.0.7

## 更新内容
config文件改名为【在线礼包.json】，并对其修改项合理汉化  
优化了计时器
